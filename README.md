# End-to-End-Data-Engineering-Pipeline-for-Sales-Analytics
Build a data pipeline that ingests raw sales data, processes it, stores it, and generates analytics using hive, apache spark and apache hadoop.
This project demonstrates an end-to-end data engineering pipeline.

Technologies Used
1. Python
2. Apache Spark
3. Hadoop
4, Hive

Pipeline Steps
1. Data ingestion
2. Data transformation
3. Data storage
4. Data analytics

Output - Category-wise revenue analysis.
